# 전 세션 수용분 전수 시각 검증 — 코덱스 교차 검증용 (2차)

작성: 2026-07-21 (Claude Code). 1차 문서(`session_009_exhaustive_verification.md`, 009 한정)를 확장해
**4개 세션의 수용 pose 행 2,216개 전수**를 다룬다. 코덱스 1차 교차 검증의 지적
(완전 독립 블라인드 아님 → κ 미산출)을 반영해, 이번 문서는 **코덱스가 라벨 CSV를
열기 전에 몽타주만으로 독립 라벨링**할 수 있도록 자료를 분리 배치했다.

## 1. 방법 (1차와 동일 + 보강)

1. 프레임당 1셀 블라인드 몽타주(빨간 X=대체된 cleaned, 초록 O=수용, 랜드마크 약어만).
   006/007/008은 4×4 그리드(총 37장), 009는 3×3(11장, 1차 산출물).
2. **기계 사전 스크리닝 추가**: 수용 좌표가 같은 프레임의 유효 cleaned 랜드마크
   어디에서도 0.06 이상 떨어진 행을 자동 플래그(9행) → 전건 고배율 줌 재판독.
3. 몽타주 판독 중 의심 셀은 개별 줌 재판독 (예: 008 f1750 — 몽타주 약어 오독을
   줌으로 교정한 사례, 실제로는 핸드스탠드의 바닥 짚은 right_index 정답).

1. **가드·margin 변경 불필요 재확정** (1차 결론의 4세션 확장): 걸러낼 오류가

## 5. 코덱스 2차 교차 검증 요청 (κ 프로토콜)

1. **독립 블라인드 라벨링**: 라벨 CSV와 본 문서 2장을 보기 전에 몽타주 48장만으로
   행별 3단계 라벨을 먼저 기록할 것 (κ 산출의 전제).
2. 라벨 완료 후 CSV와 대조해 3×3 혼동행렬 + Cohen's κ 보고 (오류 0건일 때의
   κ 불안정성 명시 포함 — 1차 지적사항).
3. 기계 플래그 로직(0.06 스켈레톤 거리)의 타당성 검토 — 더 나은 사전 스크리닝
   기준이 있으면 제안.
4. "가드 변경 불필요" 재확정의 반례 탐색.
